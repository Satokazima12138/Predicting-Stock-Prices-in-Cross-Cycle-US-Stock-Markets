# Predicting-Stock-Prices-in-Cross-Cycle-US-Stock-Markets
Official repository for my undergraduate dissertation: "Predicting Stock Prices in Cross-Cycle US Stock Markets: An Attribution Analysis from Statistical Models to Deep Hybrid Architectures". This database contains the full implementation of StockMixer, NYSE data restoration algorithms, and comprehensive backtesting results across NASDAQ, NYSE, and S&amp;P 500 (2013-2022).

## Code and Data Availability
Due to the large size of the project files, the source code, implementation details, trained model components, and related datasets are stored in Google Drive rather than directly in this GitHub repository.

The code and data can be accessed through the following link:
https://drive.google.com/drive/folders/1H0nqBabzVPVFd5_c9NG7KNVdGHBU86iK?usp=drive_link

Please note that some file paths in the scripts may need to be modified before running the code. The original paths were set according to my local project folder structure, so users should update the dataset and output directories to match their own environment. Otherwise, path-related errors may occur during execution.
